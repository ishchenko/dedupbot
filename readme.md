# Telegram dedupbot

[@dedupbot](https://t.me/dedupbot)

Checks for duplicate `file_id` in [Document](https://core.telegram.org/bots/api#document) and rants. Useful for groups that strive for unique [@gif](https://t.me/gif) content.
